# JanaGana Help Content

This file contains all help content for the JanaGana documentation system.

## Getting Started

### Sign Up & Create Organization

**Title:** Sign Up & Create Organization

**Content:**
1. Go to janagana.namasteneedham.com
2. Click "Sign Up"
3. Sign in with your email or Google account
4. Enter your organization name (e.g., "Purple Wings")
5. Choose a unique slug (e.g., "purple-wings")
6. Set your primary color
7. Click "Create Organization"

**Tips:**
- Your slug will be used in URLs and for website integration
- Choose a slug that's easy to remember
- You can change the color later in settings

---

### Complete Your Profile

**Title:** Complete Your Profile

**Content:**
After creating your organization, complete your profile:
1. Go to Settings → Organization
2. Add your organization description
3. Upload your logo
4. Set your timezone
5. Configure your contact information

**Why this matters:**
- Your profile appears on member portal
- Logo and branding are used throughout the app
- Timezone affects event times and notifications

---

### Invite Team Members

**Title:** Invite Team Members

**Content:**
1. Go to Settings → Team
2. Click "Invite Member"
3. Enter their email address
4. Select their role (Admin, Editor, Viewer)
5. Click "Send Invite"

**Roles:**
- **Admin:** Full access to all features
- **Editor:** Can manage content but not settings
- **Viewer:** Read-only access

## Integrations

### Quick Start Guide

**Title:** Website Integration Quick Start

**Content:**
Add JanaGana to your website in 3 steps:

**Step 1: Add the script**
```html
<script src="https://janagana.namasteneedham.com/janagana-embed.js"></script>
<script>
  Janagana.init({
    tenantSlug: 'your-slug',
    apiUrl: 'https://janagana.namasteneedham.com'
  });
</script>
```

**Step 2: Add a container**
```html
<div id="newsletter-widget"></div>
```

**Step 3: Initialize widget**
```html
<script>
  Janagana.newsletter('newsletter-widget');
</script>
```

That's it! See the full guide for more widgets.

## API

### API Overview

**Title:** API Overview

**Content:**
JanaGana provides REST APIs for:
- Contacts, Activities, Deals, Tasks
- Events, Registrations
- Members, Donations
- Volunteer Opportunities

**Base URL:**
`https://janagana.namasteneedham.com/api`

**Authentication:**
- Use API keys from Settings → API Keys
- Include in header: `X-API-Key: your-key`

**Rate limits:**
- 100 requests per minute
- 10,000 requests per day

---

### Authentication

**Title:** API Authentication

**Content:**
**Get API key:**
1. Go to Settings → API Keys
2. Click "Generate API Key"
3. Copy the key
4. Store securely

**Use API key:**
```bash
curl -H "X-API-Key: your-key" \
  https://janagana.namasteneedham.com/api/dashboard/crm/contacts
```

**Security:**
- Never expose API keys in frontend code
- Use environment variables
- Rotate keys regularly

---

### Contacts API

**Title:** Contacts API

**Content:**
**List contacts:**
```bash
GET /api/dashboard/crm/contacts
```

**Create contact:**
```bash
POST /api/dashboard/crm/contacts
{
  "firstName": "John",
  "lastName": "Doe",
  "email": "john@example.com",
  "phone": "+1234567890"
}
```

**Update contact:**
```bash
PUT /api/dashboard/crm/contacts/{id}
{
  "firstName": "Jane",
  "lastName": "Doe"
}
```

**Delete contact:**
```bash
DELETE /api/dashboard/crm/contacts/{id}
```

---

### Events API

**Title:** Events API

**Content:**
**List events:**
```bash
GET /api/dashboard/events
```

**Create event:**
```bash
POST /api/dashboard/events
{
  "title": "Annual Gala",
  "description": "Our yearly celebration",
  "startDate": "2024-12-01T19:00:00Z",
  "location": "Main Hall",
  "priceCents": 5000
}
```

**Register for event:**
```bash
POST /api/plugin/event-registrations
{
  "eventId": "event-id",
  "memberEmail": "member@example.com"
}
```
